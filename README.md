# NeuralChat - Free AI Chat with Gemini

A beautiful AI chat interface powered by **Google Gemini** - completely free!

🌐 **Live Demo**: https://antono4.github.io/Local-AI-Like-Ollama/

## ✨ Features

- 🤖 **Google Gemini** - Powerful AI, completely free
- 💬 **Beautiful UI** - Dark futuristic theme with smooth animations
- 💾 **Auto-save** - Conversations saved locally in your browser
- ⚙️ **Customizable** - Temperature, max tokens, system prompt
- 📱 **Responsive** - Works on desktop, tablet, and mobile
- 🔒 **Privacy First** - API keys stored locally in your browser
- 🆓 **No Credit Card** - 100% free tier

## 🚀 Getting Started (3 Steps!)

### Step 1: Open the Web App
Visit: **https://antono4.github.io/Local-AI-Like-Ollama/**

### Step 2: Get Free Gemini API Key
1. Go to **https://aistudio.google.com/**
2. Sign in with your Google account
3. Click **"Get API Key"** in the sidebar
4. Click **"Create API Key"**
5. Copy the API key

### Step 3: Start Chatting!
1. Paste your API key in NeuralChat
2. Click **Save**
3. Select a model (Gemini 1.5 Flash recommended)
4. Start chatting! 💬
