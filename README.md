# NeuralChat - Free AI Chat

A simple, beautiful AI chat interface powered by **Google Gemini** - completely free!

🌐 **Live Demo**: https://antono4.github.io/Local-AI-Like-Ollama/

## ✨ Features

- 🤖 **Google Gemini** - Powerful AI, completely free
- 💬 **Clean UI** - Dark theme, simple design
- 💾 **Auto-save** - Conversations saved locally
- 📱 **Responsive** - Works on all devices
- 🔒 **Privacy First** - API keys stored locally
- 🆓 **No Credit Card** - 100% free tier

## 🚀 Quick Start (2 Steps!)

### 1. Get Free API Key
1. Go to **https://aistudio.google.com/apikey**
2. Sign in with Google
3. Click **"Create API Key"**
4. Copy the key

### 2. Start Chatting
1. Open **https://antono4.github.io/Local-AI-Like-Ollama/**
2. Paste your API key
3. Click **Start Chatting**
4. Done! 💬
